Gràcies per l'interès en contribuir a aquest projecte de Xatbot per a WordPress! Volem assegurar-nos que el procés sigui senzill, ordenat i eficient per a tots els desenvolupadors.

Al participar en aquest repositori, acceptes complir amb les següents directrius de treball:
🛠️ Procés de Desenvolupament Tècnic

    Fes un Fork del Projecte: Crea una còpia d'aquest repositori al teu compte personal abans d'aplicar canvis substancials.

    Crea una Branca de Treball (Branch): Utilitza noms clars i descriptius que facin referència a la millora que vols introduir:
    Bash

    git checkout -b feature/nova-funcio-xatbot
    # o bé per corregir errors (bugs):
    git checkout -b fix/error-connexio-backend

    Desenvolupa amb Netedat: Respecta les normes d'estil de programació (PEP 8 per al backend de Python i estàndards semàntics al frontend HTML/CSS).

📥 Tràmits de Propostes (Pull Requests)

Quan consideris que les teves modificacions estan llistes per ser validades i integrades al repositori principal:

    Assegura't d'actualitzar el fitxer CHANGELOG.md indicant breument quins canvis has afegit a la secció corresponent.

    Obre una Pull Request (PR) detallant de manera resumida el propòsit del teu codi, quins problemes soluciona i com realitzar les proves tècniques necessàries per comprovar que tot funciona correctament.

    Evita pujar en els teus commits fitxers de configuració locals modificats, dependències no registrades a requirements.txt o còpies del fitxer amb les claus reals .env.

Agraïm enormement la teva dedicació per millorar les interaccions automàtiques d'aquest bot!
